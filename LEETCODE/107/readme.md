107. 二叉树的层次遍历 II
给定一个二叉树，返回其节点值自底向上的层次遍历。 （即按从叶子节点所在层到根节点所在的层，逐层从左向右遍历）

例如：
给定二叉树 [3,9,20,null,null,15,7],

    3
   / \
  9  20
    /  \
   15   7
返回其自底向上的层次遍历为：

[
  [15,7],
  [9,20],
  [3]
]

## 题解
DFS深度优先遍历
a[dep] = a[dep] || [];
如果当前a[dep]存在，则不变；不存在则执行 a[dep] = [];

最后要记得将得到的数组a反转。a.reverse
